> Danger.js plugin for monitoring code coverage on changed files.

## Usage

Install:

```sh
yarn add danger-plugin-karma-instanbul --dev
```

At a glance:

```js
// dangerfile.js
import karmaInstanbul from 'danger-plugin-karma-instanbul'

karmaInstanbul()
```
